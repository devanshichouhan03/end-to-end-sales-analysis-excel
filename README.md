# end-to-end-sales-analysis-excel
Interactive Excel dashboard analyzing sales performance, target achievement, and executive metrics using Pivot Tables, XLOOKUP, and INDEX-MATCH.
End-to-End Sales Analysis Dashboard (Excel)

 Objective

To analyze raw sales data and build an interactive dashboard to track performance, target achievement, and executive-wise metrics.

Tools Used

1 ) Microsoft Excel

2) Pivot Tables

3) Pivot Charts

4) Slicers

5) Cleaned and structured raw dataset

6) Performed data transformation and validation

7) Applied advanced Excel formulas for automated analysis

8) Built interactive dashboard with dynamic slicers

9) Designed navigation between multiple dashboard views

Key Metrics

1) Total Sales

2) Target Hit %

3) Away from Target %
